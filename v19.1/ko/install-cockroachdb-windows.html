마지막 릴리즈, {{ page.release_info.version }} 에 추가된 내용을 알고 싶으면 릴리즈 노트를 보십시오. 오래된 버전에서 이 릴리즈로 업그레이드하려면 클러스터 업그레이드를 보십시오.

실행파일 다운로드

{{site.data.alerts.callout_danger}}Windows용 카크로치디비 실행파일은 실험적이며 프로덕션 환경에 적합하지 않습니다. Windows 8 이상이 필요합니다.{{site.data.alerts.end}}

  1. Windows용 카크로치디비 {{ page.release_info.version }} 아카이브를 다운로드받고, 압축을 푸십시오.

  2. PowerShell을 열어, 실행파일이 있는 경로로 이동한 후, 동작을 확인하십시오:

    copy
    PS C:\cockroach-{{ page.release_info.version }}.windows-6.2-amd64> .\cockroach.exe version

  3. 카크로치디비 릴리즈와 모범사례를 최신으로 유지하십시오:

쿠버네티스 사용하기

설정 파일 또는 헬름 패키지 매니저를 통해 쿠버네티스로 카크로치디비를 오케스트레이션하려면, 미니쿠베를 이용한 카크로치디비 로컬 오케스트레이션을 확인하십시오.

Use Docker

{{site.data.alerts.callout_danger}} 카크로치디비와 같은 스테이트풀 애플리케이션을 도커에서 사용하는 것은 일반적인 도커 사용보다 더 복잡하고 오류가 발생하기 쉽습니다. 도커에 매우 숙련되어 있지 않다면, 다른 설치 및 배포 방법으로 시작하시는 것을 권고드립니다.{{site.data.alerts.end}}

  1. 도커 for Windows를 설치하십시오.

    도커 for Windows는 64비트 Windows 10 Pro와 Hyper-V를 필요로 합니다. 공식 문서를 통해 세부내용을 더 확인할 수 있습니다. 시스템이 요구사항을 만족하지 않는다면, 도커 툴박스 를 이용할 수 있습니다.

  2. PowerShell을 열어 도커 데몬이 백그라운드에서 동작하는 것을 확인하십시오:

    PS C:\Users\username> docker version

    서버 목록을 확인할 수 없다면, 도커 for Windows를 실행하십시오.

  3. 로컬 드라이브를 공유하십시오. 이렇게 하면 컨테이너를 중지하거나 삭제한 후에 로컬 디렉토리를 데이터 볼륨으로 마운트하여 데이터를 유지할 수 있습니다.

  4. 카크로치디비 {{page.release_info.version}} 릴리즈를 도커 허브에서 다운로드 받으십시오:

    copy
    PS C:\Users\username> docker pull {{page.release_info.docker_image}}:{{page.release_info.version}}

  5. 카크로치디비 릴리즈와 모범사례를 최신으로 유지하십시오:
